### 0. Project definition
-  Hardware Selection : 
	-  MCU : STM32F401RE 
	-  Sensor : BME280
	-  Bus : Initially I2C, potential to use SPI after.
-  Pin mapping : 
	-  USART2
	-  PA2 -> USART2_Tx
	-  PA3-> USART2_Rx
	-  I2C1 
	-  PB8 -> I2C1_SCL
	-  PB9 -> I2C1_SDA
-  Tool chain and project firmware skeleton : 
	-  Use modules (.c/.h) pairs that have dedicated responsibility
	-  Modules : uart2, bme280, i2c1, config, systick, rcc 
-  USART bring up.
	-  Configure for printing debug statements to terminal.
