# Overview 
-  The EXTI module configures the onboard B1 button on the Nucleo-F401RE as an external interrupt source. A falling edge on PC13 fires an IRQ, which applies a debounce check and sets a print flag. Main loop polls this flag and prints the latest reading when it is set. 
# Hardware Configuration 

|   Parameter   | Configuration  |
| :-----------: | :------------: |
|   GPIO Pin    |      PC13      |
|    Trigger    |  Falling edge  |
|   Debounce    | 200ms Software |
|   EXTI Line   |    Line 13     |
|  NVIC Vector  | EXTI15_10_IRQn |
| NVIC Priority |       6        |
-  B1 is active low, falling edge represents the button being pressed and rising edge is the button being released. 
-  EXTI lines 0-15 can be sourced from any GPIO port. EXTI line 13 could be for PA13, PB13, PC13... defaults to PA13. Must explicitly route the EXTI controller to the GPIOC port using `SYSCFG->EXTICR`.    
-  The STM32F4 NVIC does not give every EXTI line its dedicated interrupt vector, lines 10-15 share to reduce table size.  
-  In the IRQ, ensure to check `PR` to confirm the correct line fired before executing anything. 
# Initialisation 
### Stage 1 - Enable Peripheral Clocks 
-  SYSCFG and GPIOC clocks are enabled before any register access, also include dummy reads. 
-  SYSCFG is on APB2 and GPIOC is on ABH1. 
### Stage 2 - Configure GPIO 
-  PC13 needs to be configured to input mode. Done by clearing respective bits in `MODER`. Only configuration that needs to occur. 
### Step 3 - Configure NVIC and EXTI
-  Four steps, routing, falling edge, unmask, NVIC.
-  Unmasking must occur after routing and falling edge else risk ISR firing before setup is complete. 
-  NVIC priority is below I2C1 but above USART2. 
# Implementation 
-  The handler must confirm that line 13 fired by checking `PR`, then clear the pending bit for else risk an interrupt storm after ISR exit. 
-  Implement debounce check, compare the current tick against the timestamp of the last valid press, if less than a configured time just exit the ISR without further action as deemed an invalid input.
-  If passes debounce check, then set the print flag high. This flag is polled in main.c and is also reset by main.c. Note this flag must be volatile as read in two different execution contexts. 
# Testing and Verification 
-  System testing and verification can occur now, simply press the button and if line is correctly printed to serial terminal then testing is passed. Further verified debounce by rapidly pressing button. 
# Design Notes 
-  **EXTI (External Interrupt Controller) -** is a core peripheral that sits between the GPIOs and the NVIC. Its job is to say "pin x" has just changed fire an interrupt. EXTI watches the configured input lines for edge transitions, when it detects the configured edge it sets a pending bit and signals the NVIC.
